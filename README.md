# Tinkering Lab by Desh

A hands-on space to learn by doing.

## Subjects
- **Physics** → `physics/` : Chapters, Interactive sims, Real experiments
- **Maths** → `maths/` : Chapters, Interactive problems, Real applications  
- **CS** → `cs/` : Chapters, Interactive code, Real projects

Each subject has 3 parts:
1. `chapters/` - Theory + notes
2. `interactive/` - Code, notebooks, web demos  
3. `experiments/` - Real-world projects + DIY

Rule: Tinker first, perfect later.
